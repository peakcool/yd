# yd

> Vonic + Webpack

### 开始
```bash
$ npm install
$ npm run dev

```


#### Mock Json 启动

```bash
$ npm run mockdev

```


### 构建
```bash
$ npm run build
```
index.html 文件head中写入:

	<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=no">

index.html 文件body中写入:

	<von-app></von-app>
